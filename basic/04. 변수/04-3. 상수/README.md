# 상수

- **상수**는 값을 절대 변경할 수 없는 값임.
- 자신이나 다른 사람이 변수의 값을 변경하지 못하게 할 때 사용함.
- 변경될 가능성이 없는 **정해진 값이나 보안 상의 이유로 변하면 안 되는 값**을 사용할 때 이용함.
- 또한 상수는 선언할 때 값을 할당해야 함.
    
    ```cpp
    // const int minutes_per_hour;
    // minutes_per_hour = 60; 오류가 생김
    const float PI = 3.14;
    cout << PI; // 3.14;
    
    // PI = 30; 오류가 생김
    ```
