# 변수

- **변수**는 데이터 값을 저장하기 위한 컨터이너임.
- **자주 사용하는 자료형 종류**
    
    
    | 구분 | 설명 |
    | --- | --- |
    | int | 정수를 저장함 |
    | double | 소수점 이하 부동 소수점 숫자를 저장함. |
    | char  | ‘a’나 ‘B’와 같은 단일 문자를 저장함. |
    | string | “안녕 세상아!”와 같은 문자열을 저장함. |
    | bool | true 또는 false의 2개의 값만 저장함. |
- 변수는 `자료형 변수명 = 값;` 의 꼴로 생성함.
    
    ```cpp
    int value = 15;
    cout << value;
    
    // 15
    ```
